---
title: "Orbit Decomposition Lemma"
description: "Orbits of a group action form a partition of the underlying set"
---

**Orbit Decomposition Lemma**: Let $G$ be a {{< knowl id="group" text="group" >}} acting on a {{< knowl id="set" section="shared-foundations" text="set" >}} $X$ via a {{< knowl id="group-action" text="group action" >}}. Then the set of {{< knowl id="orbit" text="orbits" >}} $\{G\cdot x : x\in X\}$ is a {{< knowl id="partition" section="shared-foundations" text="partition" >}} of $X$.

Equivalently, define a {{< knowl id="relation" section="shared-foundations" text="relation" >}} $\sim$ on $X$ by $x\sim y$ if there exists $g\in G$ with $g\cdot x=y$. Then $\sim$ is an {{< knowl id="equivalence-relation" section="shared-foundations" text="equivalence relation" >}}, and its {{< knowl id="equivalence-class" section="shared-foundations" text="equivalence classes" >}} are exactly the orbits.
