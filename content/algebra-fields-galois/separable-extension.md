---
title: "Separable extension"
description: "An algebraic extension L/K in which every element is separable over K."
---

**Definition.** An {{< knowl id="algebraic-extension" text="algebraic extension" >}} \(L/K\) is **separable** if every element of \(L\) is a {{< knowl id="separable-element" text="separable element" >}} over \(K\).

Equivalently (for finite extensions), \(L/K\) is separable iff it can be generated by adjoining separable elements.

**See also.** {{< knowl id="perfect-field" text="perfect field" >}}, {{< knowl id="galois-extension" text="Galois extension" >}}, {{< knowl id="inseparable-extension" text="inseparable extension" >}}.

**Examples.**
1. Every algebraic extension of \(\mathbb{Q}\) is separable (characteristic \(0\)).
2. Every finite field extension \(\mathbb{F}_{p^n}/\mathbb{F}_p\) is separable.
3. The extension \(\mathbb{F}_p(t^{1/p})/\mathbb{F}_p(t)\) is **not** separable (it is purely inseparable).
