---
title: "Transcendental field extension"
description: "An extension L/K that contains at least one element transcendental over K."
---

**Definition.** A {{< knowl id="field-extension" text="field extension" >}} \(L/K\) is **transcendental** if there exists \(\alpha\in L\) that is {{< knowl id="transcendental-element" text="transcendental over K" >}}. Equivalently, \(L/K\) is transcendental if it is **not** an {{< knowl id="algebraic-extension" text="algebraic extension" >}}.

A basic class of examples is the rational function field \(K(t)\), obtained by adjoining a transcendental element \(t\).

**See also.** {{< knowl id="fg-field-extension" text="finitely generated extensions" >}}, {{< knowl id="simple-extension" text="simple extension" >}}.

**Examples.**
1. \(K(t)/K\) is transcendental, with \(t\) transcendental over \(K\).
2. \(\mathbb{R}/\mathbb{Q}\) is transcendental, since \(\mathbb{R}\) contains transcendental numbers (e.g. \(\pi\)).
3. \(\mathbb{F}_p(t)/\mathbb{F}_p\) is transcendental.
