---
title: "Graph of a function"
description: "The set of ordered pairs consisting of each input and its output"
---

A **graph of a function** is the set of inputâ€“output pairs: if $f:A\to B$ is a {{< knowl id="function" text="function" >}}, then its graph is
$$
\Gamma_f=\{(a,f(a)):\ a\in A\}\subseteq A\times B.
$$

The graph is a {{< knowl id="subset" text="subset" >}} of the {{< knowl id="cartesian-product" text="Cartesian product" >}} and consists of {{< knowl id="ordered-pair" text="ordered pairs" >}}. Viewing a function as a special kind of {{< knowl id="relation" text="relation" >}}, the graph is exactly the corresponding relation.

**Examples:**
- If $f:\mathbb{R}\to\mathbb{R}$ is $f(x)=x^2$, then $\Gamma_f=\{(x,x^2):x\in\mathbb{R}\}$.
- If $A$ is any set, the graph of $\mathrm{id}_A$ is $\{(a,a):a\in A\}$.
