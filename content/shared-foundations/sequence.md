---
title: "Sequence"
description: "A function from the natural numbers to a set."
---

A **sequence** in a {{< knowl id="set" text="set" >}} $X$ is a {{< knowl id="function" text="function" >}} $a:\mathbb{N}\to X$, where $\mathbb{N}$ is the {{< knowl id="natural-numbers" text="natural numbers" >}}. One writes $a_n$ for $a(n)$, so the sequence is denoted $(a_n)_{n\in\mathbb{N}}$.

Thus a sequence is a function with {{< knowl id="domain" text="domain" >}} $\mathbb{N}$ and {{< knowl id="codomain" text="codomain" >}} $X$, and it can be manipulated using standard function constructions such as {{< knowl id="restriction-of-a-function" text="restriction" >}} and {{< knowl id="composition" text="composition" >}}.

**Examples:**
- The rule $a_n = 1/(n+1)$ defines a sequence of {{< knowl id="rational-numbers" text="rational numbers" >}}.
- The rule $a_n=7$ for all $n\in\mathbb{N}$ defines a constant sequence in $\mathbb{Z}$ (the {{< knowl id="integers" text="integers" >}}).
