---
title: "Power set"
description: "The set of all subsets of a given set."
---

A **power set** of a set $A$ is the set
$$
\mathcal{P}(A)=\{B : B\subseteq A\}.
$$

Thus $\mathcal{P}(A)$ collects all {{< knowl id="subset" text="subsets" >}} of $A$ into a single {{< knowl id="set" text="set" >}}. Power sets are central when forming collections of sets, such as families indexed by an index set.

**Examples:**
- If $A=\varnothing$, then $\mathcal{P}(A)=\{\varnothing\}$.
- If $A=\{1,2\}$, then $\mathcal{P}(A)=\{\varnothing,\{1\},\{2\},\{1,2\}\}$.
