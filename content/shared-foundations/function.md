---
title: "Function"
description: "A relation that assigns each input exactly one output"
---

A **function** is a {{< knowl id="relation" text="relation" >}} $f\subseteq A\times B$ between {{< knowl id="set" text="sets" >}} $A$ and $B$ such that for every $a\in A$ there exists a unique $b\in B$ with $(a,b)\in f$; we write this $b$ as $f(a)$ and denote the function by $f:A\to B$.

The set $A$ is called the {{< knowl id="domain" text="domain" >}} and the set $B$ the {{< knowl id="codomain" text="codomain" >}}. Functions allow you to form {{< knowl id="image" text="images" >}} and {{< knowl id="preimage" text="preimages" >}} of subsets and to build new functions via {{< knowl id="composition" text="composition" >}}.

**Examples:**
- The squaring map $f:\mathbb{N}\to\mathbb{N}$ given by $f(n)=n^2$ is a function on the {{< knowl id="natural-numbers" text="natural numbers" >}}.
- If $S$ is a {{< knowl id="subset" text="subset" >}} of a set $A$, the inclusion map $i:S\to A$ defined by $i(s)=s$ is a function.
