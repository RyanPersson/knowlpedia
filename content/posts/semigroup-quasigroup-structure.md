## Definitions

- {{< knowl id="magma" section="algebra-groups" text="Magma" >}} — a set with a binary operation
- {{< knowl id="semigroup" section="algebra-groups" text="Semigroup" >}} — associative magma
- {{< knowl id="unital-magma" section="algebra-groups" text="Unital Magma" >}} — magma with identity
- {{< knowl id="monoid" section="algebra-groups" text="Monoid" >}} — semigroup with identity
- {{< knowl id="quasigroup" section="algebra-groups" text="Quasigroup" >}} — magma with division
- {{< knowl id="loop" section="algebra-groups" text="Loop" >}} — quasigroup with identity
- {{< knowl id="group" section="algebra-groups" text="Group" >}} — monoid with inverses (equivalently: associative loop)
