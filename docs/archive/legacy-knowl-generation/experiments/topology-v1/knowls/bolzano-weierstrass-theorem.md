---
title: "Bolzano-Weierstrass theorem"
description: "Every bounded sequence in Euclidean space has a convergent subsequence."
---

**Bolzano-Weierstrass theorem**: In {{< knowl id="euclidean-space" text="Euclidean space" >}} $\mathbb{R}^n$, every sequence whose range is {{< knowl id="bounded-set" text="bounded" >}} has a {{< knowl id="convergent-sequence" text="convergent sequence" >}} subsequence.

A common equivalent form is: every infinite bounded subset of $\mathbb{R}^n$ has a limit point. This result is closely connected to compactness: by the {{< knowl id="heine-borel-theorem" text="Heine-Borel theorem" >}}, closed and bounded subsets of $\mathbb{R}^n$ are {{< knowl id="compact-set" text="compact" >}}, and in metric spaces {{< knowl id="sequential-compactness-equals-compactness" text="compactness is equivalent to sequential compactness" >}}.

**Proof sketch** *(optional, include for major theorems)*:  
A bounded sequence in $\mathbb{R}^n$ lies inside some closed ball. In $\mathbb{R}^n$ this ball is compact (Heine-Borel), so it is sequentially compact; hence the sequence has a convergent subsequence.
