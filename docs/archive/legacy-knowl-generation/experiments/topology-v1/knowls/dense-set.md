---
title: "Dense set"
description: "A subset whose closure is the entire space."
---

Let $(X,\tau)$ be a {{< knowl id="topological-space" text="topological space" >}} and let $A\subseteq X$. The set $A$ is **dense** in $X$ if its {{< knowl id="closure" text="closure" >}} equals $X$.

Equivalently, $A$ is dense iff every nonempty {{< knowl id="open-set" text="open set" >}} intersects $A$ (i.e., has nonempty {{< knowl id="intersection" text="intersection" >}} with $A$). Dense sets capture the idea that $A$ comes arbitrarily close to every point of the space.

**Examples:**
- In $\mathbb{R}$ with the usual topology, $\mathbb{Q}$ is dense.
- In a discrete topology on $X$, the only dense subset is $X$ itself.
- If $X$ is nonempty, then $\varnothing$ is not dense.
