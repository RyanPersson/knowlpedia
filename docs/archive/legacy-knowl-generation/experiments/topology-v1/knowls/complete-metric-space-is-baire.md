**Complete metric space is Baire**: Every {{< knowl id="complete-metric-space" text="complete metric space" >}} is a {{< knowl id="baire-space" text="Baire space" >}}.

This is an immediate reformulation of the {{< knowl id="baire-category-theorem" text="Baire category theorem" >}}: the theorem asserts exactly the defining “countable intersection of open dense sets is dense” property. In practice one often applies it in the explicit form {{< knowl id="intersection-of-dense-open-is-dense" text="intersection of dense open sets is dense" >}}.
