---
title: "Path"
description: "A continuous map from the unit interval into a topological space"
---

A **path** in a {{< knowl id="topological-space" text="topological space" >}} $X$ is a {{< knowl id="continuous-map" text="continuous map" >}} $\gamma:[0,1]\to X$. The points $\gamma(0)$ and $\gamma(1)$ are called the initial and terminal points (or endpoints) of the path.

A path is a special case of a {{< knowl id="curve" text="curve" >}} (with domain restricted to the unit interval). Paths are the basic objects used to define {{< knowl id="path-connected-set" text="path connectedness" >}}.

**Examples:**
- In {{< knowl id="euclidean-space" text="Euclidean space" >}} $\mathbb{R}^n$, the straight-line map $\gamma(t)=(1-t)a+tb$ is a path from $a$ to $b$.
- The constant map $\gamma(t)=x_0$ is a path from $x_0$ to $x_0$ in any space.
- On the circle in $\mathbb{R}^2$, $\gamma(t)=(\cos(2\pi t),\sin(2\pi t))$ is a closed path (it starts and ends at the same point).
