---
title: "Nowhere dense set"
description: "A subset whose closure has empty interior"
---

A **nowhere dense set** in a {{< knowl id="topological-space" text="topological space" >}} $X$ is a subset $A\subseteq X$ such that the {{< knowl id="closure" text="closure" >}} $\overline{A}$ has empty {{< knowl id="interior" text="interior" >}}, i.e. $\operatorname{int}(\overline{A})=\varnothing$.

Nowhere dense sets are the basic building blocks of {{< knowl id="meager-set" text="meager sets" >}}. In particular, a space is a {{< knowl id="baire-space" text="Baire space" >}} precisely when countably many nowhere dense sets cannot cover any nonempty open set.

**Examples:**
- In $\mathbb{R}$ with its usual topology, the integers $\mathbb{Z}$ are nowhere dense.
- The Cantor set in $\mathbb{R}$ is nowhere dense.
- No nonempty {{< knowl id="open-set" text="open set" >}} is nowhere dense.
