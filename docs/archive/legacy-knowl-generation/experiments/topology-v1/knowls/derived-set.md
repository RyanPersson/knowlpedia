---
title: "Derived set"
description: "The set of all limit points of a subset."
---

Let $(X,\tau)$ be a topological space and let $A\subseteq X$. The **derived set** of $A$, denoted $A'$, is the set of all {{< knowl id="limit-point-accumulation-point-cluster-point" text="limit points" >}} of $A$.

The derived set isolates the “accumulation behavior” of $A$. In a {{< knowl id="t1-space" text="T1 space" >}}, one has $\overline{A}=A\cup A'$, linking it directly to {{< knowl id="closure" text="closure" >}}.

**Examples:**
- If $A=\{1/n : n\in\mathbb{N}\}\subseteq \mathbb{R}$, then $A'=\{0\}$ in the usual topology.
- If $A$ is finite in $\mathbb{R}$, then $A'=\varnothing$.
- If $A$ is dense in $\mathbb{R}$ (for instance, $\mathbb{Q}$), then $A'=\mathbb{R}$.
