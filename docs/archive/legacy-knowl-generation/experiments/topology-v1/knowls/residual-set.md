---
title: "Residual set"
description: "A set whose complement is meager"
---

A **residual set** (or *comeager set*) in a {{< knowl id="topological-space" text="topological space" >}} $X$ is a subset $R\subseteq X$ whose {{< knowl id="complement" text="complement" >}} $X\setminus R$ is {{< knowl id="meager-set" text="meager" >}}.

Equivalently, $R$ contains a {{< knowl id="countable-set" text="countable" >}} {{< knowl id="intersection" text="intersection" >}} of {{< knowl id="open-set" text="open" >}} {{< knowl id="dense-set" text="dense" >}} sets. In a {{< knowl id="baire-space" text="Baire space" >}}, every residual set is dense.

**Examples:**
- In $\mathbb{R}$, the irrational numbers form a residual set because $\mathbb{Q}$ is meager.
- In any space, the intersection of countably many open dense sets is residual.
- A residual set need not be open (for example, the irrationals in $\mathbb{R}$ are not open).
