---
title: "Complete metric space"
description: "A metric space in which every Cauchy sequence converges."
---

A **complete metric space** is a {{< knowl id="metric-space" text="metric space" >}} $(X,d)$ such that every {{< knowl id="cauchy-sequence" text="Cauchy sequence" >}} in $X$ is {{< knowl id="convergent-sequence" text="convergent" >}} to a point of $X$.

Completeness is a foundational hypothesis in many results; for example, it underlies the {{< knowl id="cantor-intersection-theorem" text="Cantor intersection theorem" >}} and many fixed-point principles.

**Examples:**
- $\mathbb{R}^n$ with the Euclidean metric is complete.
- $\mathbb{Q}$ with the usual metric is not complete.
- Any {{< knowl id="banach-space" text="Banach space" >}} becomes a complete metric space via the metric induced by its norm.
