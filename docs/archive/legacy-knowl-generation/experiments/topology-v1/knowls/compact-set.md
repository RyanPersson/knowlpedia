---
title: "Compact set"
description: "A set for which every open cover has a finite subcover"
---

A **compact set** $K$ in a {{< knowl id="topological-space" text="topological space" >}} $X$ is a subset such that every {{< knowl id="open-cover" text="open cover" >}} of $K$ contains a finite subcover.

Compactness has several equivalent characterizations. One important reformulation is the {{< knowl id="finite-intersection-property-theorem" text="finite intersection property" >}}. In {{< knowl id="metric-space" text="metric spaces" >}}, compactness can also be checked via sequences (see {{< knowl id="sequential-compactness-equals-compactness" text="sequential compactness equals compactness" >}}). In {{< knowl id="euclidean-space" text="Euclidean space" >}}, the {{< knowl id="heine-borel-theorem" text="Heine–Borel theorem" >}} gives a concrete closed-and-bounded criterion.

**Examples:**
- The interval $[0,1]\subset\mathbb{R}$ is compact.
- Any finite subset of any topological space is compact.
- The open interval $(0,1)\subset\mathbb{R}$ is not compact (e.g. it is covered by $\{(1/n,\,1)\}_{n\ge 2}$ but no finite subcollection covers it).
