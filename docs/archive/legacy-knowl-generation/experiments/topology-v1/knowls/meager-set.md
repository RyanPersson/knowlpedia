---
title: "Meager set"
description: "A set that is a countable union of nowhere dense sets"
---

A **meager set** (or *set of first category*) in a {{< knowl id="topological-space" text="topological space" >}} $X$ is a subset $A\subseteq X$ that can be written as a {{< knowl id="countable-set" text="countable" >}} {{< knowl id="union" text="union" >}} $A=\bigcup_{n\ge 1} A_n$ where each $A_n$ is {{< knowl id="nowhere-dense-set" text="nowhere dense" >}} in $X$.

Meager sets are “small” in the sense of category: their complements are {{< knowl id="residual-set" text="residual" >}} sets, and in a {{< knowl id="baire-space" text="Baire space" >}} no nonempty open set can be meager.

**Examples:**
- The rational numbers $\mathbb{Q}\subset \mathbb{R}$ are meager in $\mathbb{R}$ (they are a countable union of singletons).
- Any nowhere dense subset of a space is meager.
- A set can be meager while being {{< knowl id="dense-set" text="dense" >}} (for example, $\mathbb{Q}$ is dense in $\mathbb{R}$).
