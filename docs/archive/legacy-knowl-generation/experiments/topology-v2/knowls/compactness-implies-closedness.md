---
title: "Compactness implies closedness"
description: "In a metric space, every compact set is closed"
---

**Compactness implies closedness:** If $(X,d)$ is a {{< knowl id="metric-space" section="topology-metric" text="metric space" >}} and $K\subseteq X$ is {{< knowl id="compact-set" text="compact" >}}, then $K$ is {{< knowl id="closed-set" section="topology-core" text="closed" >}} in $X$.

This is a special case of {{< knowl id="compact-subset-of-hausdorff-is-closed" text="compact subsets of Hausdorff spaces are closed" >}}, since every metric space is {{< knowl id="hausdorff-space" section="topology-separation" text="Hausdorff" >}}.
