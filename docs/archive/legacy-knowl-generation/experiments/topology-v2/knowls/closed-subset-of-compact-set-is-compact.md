title: "Closed subset of a compact set is compact"

---

**Closed subset of a compact set is compact:** Let $X$ be a {{< knowl id="topological-space" section="topology-core" text="topological space" >}} and let $K\subseteq X$ be {{< knowl id="compact-set" text="compact" >}}. If $F\subseteq K$ is {{< knowl id="closed-set" section="topology-core" text="closed" >}} (in the subspace topology on $K$), then $F$ is compact.

This permanence property is frequently combined with {{< knowl id="continuous-image-of-compact-set-is-compact" text="continuous images of compact sets" >}}.
