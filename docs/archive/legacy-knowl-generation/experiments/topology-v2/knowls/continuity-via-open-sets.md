---
title: "Continuity via open sets"
description: "A map is continuous exactly when preimages of open sets are open."
---

**Continuity via open sets:** Let $f:X\to Y$ be a {{< knowl id="function" section="shared-foundations" text="function" >}} between {{< knowl id="topological-space" text="topological spaces" >}}. Then $f$ is {{< knowl id="continuous-map" text="continuous" >}} if and only if for every {{< knowl id="open-set" text="open set" >}} $V\subseteq Y$, the {{< knowl id="preimage" section="shared-foundations" text="preimage" >}} $f^{-1}(V)\subseteq X$ is open in $X$.

Equivalently, $f$ is continuous if and only if the preimage of every closed set in $Y$ is closed in $X$.
