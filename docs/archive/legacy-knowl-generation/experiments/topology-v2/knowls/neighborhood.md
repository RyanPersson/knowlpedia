---
title: "Neighborhood"
description: "A set that contains an open set around a point."
---

A **neighborhood** of a point $x\in X$ in a {{< knowl id="topological-space" text="topological space" >}} is a subset $N\subseteq X$ for which there exists an {{< knowl id="open-set" text="open set" >}} $U$ such that $x\in U\subseteq N$.

Neighborhoods capture “local” information around points and are used to define {{< knowl id="continuous-map" text="continuity" >}} in terms of images of neighborhoods.

**Examples:**
- In $\mathbb{R}$, for $\varepsilon>0$ the interval $(x-\varepsilon,x+\varepsilon)$ is a neighborhood of $x$.
- Every open set containing $x$ is a neighborhood of $x$.
- In $\mathbb{R}$ with the usual topology, $\{0\}$ is not a neighborhood of $0$.
