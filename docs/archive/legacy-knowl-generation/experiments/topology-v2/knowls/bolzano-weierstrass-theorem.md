---
title: "Bolzano–Weierstrass theorem"
description: "Every bounded sequence in Euclidean space has a convergent subsequence"
---

**Bolzano–Weierstrass theorem:** In $\mathbb{R}^n$ with the Euclidean metric (from the {{< knowl id="euclidean-norm" section="linear-algebra" text="Euclidean norm" >}}), every {{< knowl id="bounded-set" text="bounded" >}} sequence has a {{< knowl id="convergent-sequence" text="convergent" >}} subsequence.

This is a foundational compactness principle: it can be viewed as a sequential form of {{< knowl id="heine-borel-theorem" section="topology-compactness" text="Heine–Borel" >}} and is closely tied to {{< knowl id="sequentially-compact-set" section="topology-compactness" text="sequential compactness" >}}.
