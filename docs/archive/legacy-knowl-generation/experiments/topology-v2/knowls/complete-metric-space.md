---
title: "Complete metric space"
description: "A metric space in which every Cauchy sequence converges"
---

A **complete metric space** is a {{< knowl id="metric-space" text="metric space" >}} $(X,d)$ in which every {{< knowl id="cauchy-sequence" text="Cauchy sequence" >}} is a {{< knowl id="convergent-sequence" text="convergent sequence" >}} (with its limit lying in $X$).

Completeness is preserved under {{< knowl id="isometry" text="isometries" >}} (in particular, bijective isometries). Complete metric spaces also satisfy strong topological largeness properties such as being {{< knowl id="baire-space" section="topology-baire" text="Baire spaces" >}}; see {{< knowl id="complete-metric-space-is-baire" section="topology-baire" text="complete metric space is Baire" >}}.

**Examples:**
- $\mathbb{R}^n$ with the Euclidean metric is complete.
- $\mathbb{Q}$ with the usual metric is not complete.
- The open interval $(0,1)$ with the usual metric is not complete (for example, a sequence converging to $0$ in $\mathbb{R}$ need not converge in $(0,1)$).
- A {{< knowl id="banach-space" section="linear-algebra" text="Banach space" >}} is a complete normed vector space, hence a complete metric space via its norm metric.
