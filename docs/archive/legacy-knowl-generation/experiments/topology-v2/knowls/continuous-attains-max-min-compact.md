---
title: "Continuous function attains max and min on compact set"
description: "A continuous real-valued function on a compact set achieves its maximum and minimum"
---

**Extreme value theorem (compact version):** Let $K$ be a {{< knowl id="compact-set" text="compact set" >}} and let $f:K\to\mathbb R$ be a {{< knowl id="continuous-map" section="topology-core" text="continuous map" >}} (viewing $\mathbb R$ with its usual topology). Then there exist points $x_{\min},x_{\max}\in K$ such that
\[
f(x_{\min}) \le f(x) \le f(x_{\max}) \quad \text{for all } x\in K.
\]

This is a standard consequence of {{< knowl id="continuous-image-of-compact-set-is-compact" text="compactness of the image" >}} together with order properties of $\mathbb R$.
