---
title: "Compactness implies completeness"
description: "In metric spaces, compact sets are complete"
---

**Compactness implies completeness:** If $(X,d)$ is a {{< knowl id="metric-space" section="topology-metric" text="metric space" >}} and $K\subseteq X$ is {{< knowl id="compact-set" text="compact" >}}, then $K$ is complete with the subspace metric; equivalently, every {{< knowl id="cauchy-sequence" section="topology-metric" text="Cauchy sequence" >}} in $K$ converges to a point of $K$.

Combined with {{< knowl id="compactness-implies-total-boundedness" text="total boundedness" >}}, this yields a common compactness test in metric spaces.
