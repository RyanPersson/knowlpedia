---
title: "Meager set"
description: "A set that is a countable union of nowhere dense sets"
---

A **meager set** (or set of first category) in a {{< knowl id="topological-space" section="topology-core" text="topological space" >}} $X$ is a subset $M\subseteq X$ that can be written as $M=\bigcup_{n\in\mathbb{N}} A_n$ with each $A_n$ {{< knowl id="nowhere-dense-set" text="nowhere dense" >}} in $X$. In other words, $M$ is a {{< knowl id="countable-set" section="shared-foundations" text="countable" >}} {{< knowl id="union" section="shared-foundations" text="union" >}} of nowhere dense sets.

Meager sets are the “small” sets in Baire category; their complements are {{< knowl id="residual-set" text="residual sets" >}}.

**Examples:**
- In $\mathbb{R}$, the rational numbers $\mathbb{Q}$ form a meager set (as a countable union of singletons).
- In $\mathbb{R}$, every countable subset is meager.
- The space $\mathbb{Q}$ (with the subspace topology from $\mathbb{R}$) is meager in itself, so it is not a {{< knowl id="baire-space" text="Baire space" >}}.
