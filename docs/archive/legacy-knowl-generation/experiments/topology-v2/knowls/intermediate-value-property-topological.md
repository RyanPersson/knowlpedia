---
title: "Intermediate value property for continuous maps"
description: "Continuous maps from connected spaces to the real line take all intermediate values"
---

**Intermediate value property (topological form):** Let $X$ be a {{< knowl id="connected-set" text="connected" >}} {{< knowl id="topological-space" section="topology-core" text="topological space" >}} and let $f:X\to \mathbb{R}$ be a {{< knowl id="continuous-map" section="topology-core" text="continuous map" >}}. If $x_0,x_1\in X$ and a real number $y$ lies between $f(x_0)$ and $f(x_1)$, then there exists $x\in X$ such that $f(x)=y$.

Equivalently, the set $f(X)$ is an interval in $\mathbb{R}$. This is an immediate consequence of {{< knowl id="continuous-image-of-connected-set-is-connected" text="continuous images of connected sets being connected" >}} together with {{< knowl id="connected-subsets-of-r-are-intervals" text="connected subsets of R being intervals" >}}.
