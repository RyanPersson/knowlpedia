---
title: "Heine–Borel theorem"
description: "In Euclidean space, compact sets are exactly the closed and bounded sets"
---

**Heine–Borel theorem:** Let $K\subseteq \mathbb R^n$ with the metric coming from the {{< knowl id="euclidean-norm" section="linear-algebra" text="Euclidean norm" >}} (equivalently, the usual topology on {{< knowl id="euclidean-space" section="linear-algebra" text="Euclidean space" >}}). Then $K$ is {{< knowl id="compact-set" text="compact" >}} if and only if $K$ is {{< knowl id="closed-set" section="topology-core" text="closed" >}} and {{< knowl id="bounded-set" section="topology-metric" text="bounded" >}}.

This provides a concrete geometric criterion for compactness in $\mathbb R^n$.
