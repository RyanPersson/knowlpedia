---
title: "Compact set"
description: "A set for which every open cover has a finite subcover"
---

A **compact set** $K$ in a {{< knowl id="topological-space" section="topology-core" text="topological space" >}} $X$ is a set such that every {{< knowl id="open-cover" text="open cover" >}} of $K$ has a finite subcover.

Compactness is a central finiteness condition in topology; in {{< knowl id="metric-space" section="topology-metric" text="metric spaces" >}} it interacts strongly with sequential behavior via {{< knowl id="sequentially-compact-set" text="sequential compactness" >}}.

**Examples:**
- The interval $[0,1]\subset \mathbb R$ is compact (see the {{< knowl id="heine-borel-theorem" text="Heine–Borel theorem" >}}).
- Any finite subset of any topological space is compact.
- In $\mathbb R$, the open interval $(0,1)$ is not compact.
