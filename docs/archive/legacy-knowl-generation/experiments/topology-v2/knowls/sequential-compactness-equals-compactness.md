---
title: "Sequential compactness equals compactness (metric spaces)"
description: "In metric spaces, compactness is equivalent to sequential compactness"
---

**Sequential compactness equals compactness (metric spaces):** Let $(X,d)$ be a {{< knowl id="metric-space" section="topology-metric" text="metric space" >}} and let $K\subseteq X$. Then $K$ is {{< knowl id="compact-set" text="compact" >}} if and only if $K$ is {{< knowl id="sequentially-compact-set" text="sequentially compact" >}}.

Outside the metric setting, sequential compactness and compactness can differ; this equivalence is a specifically metric phenomenon.
