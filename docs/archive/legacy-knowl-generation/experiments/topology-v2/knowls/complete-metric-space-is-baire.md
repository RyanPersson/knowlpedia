**Complete metric space is Baire:** Every {{< knowl id="complete-metric-space" section="topology-metric" text="complete metric space" >}} (with its {{< knowl id="metric-induced-topology" section="topology-metric" text="metric-induced topology" >}}) is a {{< knowl id="baire-space" text="Baire space" >}}.

This is a standard formulation of the {{< knowl id="baire-category-theorem" text="Baire Category Theorem" >}} and is a common input to {{< knowl id="category-argument-template" text="category arguments" >}} producing {{< knowl id="residual-set" text="residual" >}} subsets.

**Key insight:** One constructs nested {{< knowl id="open-ball" section="topology-metric" text="open balls" >}} $B(x_n,r_n)$ with $r_n\to 0$ inside prescribed dense open sets; completeness provides a limit point lying in all of them.
