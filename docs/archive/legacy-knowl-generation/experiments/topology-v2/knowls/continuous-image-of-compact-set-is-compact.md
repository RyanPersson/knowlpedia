---
title: "Continuous image of a compact set is compact"
description: "A continuous map sends compact sets to compact sets"
---

**Continuous image of a compact set is compact:** Let $f:X\to Y$ be a {{< knowl id="continuous-map" section="topology-core" text="continuous map" >}} between {{< knowl id="topological-space" section="topology-core" text="topological spaces" >}}. If $K\subseteq X$ is {{< knowl id="compact-set" text="compact" >}}, then $f(K)\subseteq Y$ is compact.

This is one of the most-used tools for transporting compactness through constructions, and it underlies results like {{< knowl id="continuous-attains-max-min-compact" text="attainment of maxima and minima" >}}.
