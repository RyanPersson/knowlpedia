---
title: "Nowhere dense set"
description: "A subset whose closure has empty interior"
---

A **nowhere dense set** in a {{< knowl id="topological-space" section="topology-core" text="topological space" >}} $X$ is a subset $A\subseteq X$ such that $\operatorname{int}(\overline{A})=\varnothing$, where $\overline{A}$ is the {{< knowl id="closure" section="topology-core" text="closure" >}} of $A$ and $\operatorname{int}$ denotes the {{< knowl id="interior" section="topology-core" text="interior" >}}.

Nowhere dense sets are the basic building blocks of {{< knowl id="meager-set" text="meager sets" >}}; in a {{< knowl id="baire-space" text="Baire space" >}}, they are “small” in the sense that they cannot contain any nonempty {{< knowl id="open-set" section="topology-core" text="open set" >}}.

**Examples:**
- In $\mathbb{R}$ with the usual topology, any singleton $\{x\}$ is nowhere dense.
- The Cantor set in $\mathbb{R}$ is nowhere dense.
- A {{< knowl id="dense-set" section="topology-core" text="dense" >}} subset of a nonempty space is not nowhere dense (its closure is all of $X$).
