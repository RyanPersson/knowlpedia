---
title: "Continuous images of connected sets are connected"
description: "The image of a connected set under a continuous map is connected"
---

**Continuous image of a connected set is connected:** Let $f:X\to Y$ be a {{< knowl id="continuous-map" section="topology-core" text="continuous map" >}} between {{< knowl id="topological-space" section="topology-core" text="topological spaces" >}}. If $A\subseteq X$ is {{< knowl id="connected-set" text="connected" >}}, then $f(A)\subseteq Y$ is connected.

Here $f(A)$ is the {{< knowl id="image" section="shared-foundations" text="image" >}} of $A$ under $f$. This principle is a standard tool for transferring connectedness through constructions and underlies the {{< knowl id="intermediate-value-property-topological" text="intermediate value property" >}} for real-valued continuous maps.
