---
title: "Totally bounded implies Cauchy subsequence"
description: "Every sequence in a totally bounded set has a Cauchy subsequence"
---

**Totally bounded implies Cauchy subsequence:** Let $(X,d)$ be a {{< knowl id="metric-space" section="topology-metric" text="metric space" >}} and let $A\subseteq X$ be {{< knowl id="totally-bounded-set" text="totally bounded" >}}. Then every sequence in $A$ has a {{< knowl id="cauchy-sequence" section="topology-metric" text="Cauchy" >}} subsequence.

Together with completeness, this is a main route to compactness via {{< knowl id="compact-iff-complete-totally-bounded" text="complete + totally bounded" >}}.
