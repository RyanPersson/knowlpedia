---
title: "Every Cauchy sequence is bounded"
description: "In a metric space, every Cauchy sequence stays within some bounded set"
---

**Every Cauchy sequence is bounded:** Let $(X,d)$ be a {{< knowl id="metric-space" text="metric space" >}} and let $(x_n)$ be a {{< knowl id="cauchy-sequence" text="Cauchy sequence" >}} in $X$. Then the set $\{x_n : n\in\mathbb{N}\}$ is a {{< knowl id="bounded-set" text="bounded set" >}} in $X$.

This fact is often paired with {{< knowl id="convergent-sequence-is-cauchy" text="convergent sequences are Cauchy" >}} when comparing convergence and completeness.
