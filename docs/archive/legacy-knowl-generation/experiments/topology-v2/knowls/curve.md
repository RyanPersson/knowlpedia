---
title: "Curve"
description: "A continuous map from an interval of the real line into a topological space"
---

A **curve** in a {{< knowl id="topological-space" section="topology-core" text="topological space" >}} $X$ is a {{< knowl id="continuous-map" section="topology-core" text="continuous map" >}} $\gamma:I\to X$ where $I\subseteq \mathbb{R}$ is an interval (commonly $[a,b]$, $(a,b)$, or $[0,\infty)$).

Curves generalize {{< knowl id="path" text="paths" >}}: a path is a curve with domain $[0,1]$, and any curve on a closed interval $[a,b]$ can be reparameterized to a path by composing with a suitable homeomorphism $[0,1]\to[a,b]$.

**Examples:**
- The parabola in $\mathbb{R}^2$ given by $\gamma(t)=(t,t^2)$ on $I=\mathbb{R}$.
- A circular curve $\gamma(t)=(\cos t,\sin t)$ on $I=[0,2\pi]$ in $\mathbb{R}^2$.
- A constant curve $\gamma(t)=x_0$ on any interval $I$.
