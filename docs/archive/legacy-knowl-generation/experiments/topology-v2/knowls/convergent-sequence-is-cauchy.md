---
title: "Convergent sequence is Cauchy"
description: "Any convergent sequence in a metric space satisfies the Cauchy condition"
---

**Convergent sequence is Cauchy:** Let $(X,d)$ be a {{< knowl id="metric-space" text="metric space" >}}. If a sequence $(x_n)$ {{< knowl id="convergent-sequence" text="converges" >}} to some $x\in X$, then $(x_n)$ is a {{< knowl id="cauchy-sequence" text="Cauchy sequence" >}}.

In a {{< knowl id="complete-metric-space" text="complete metric space" >}}, the converse implication holds by definition: Cauchy sequences converge.
