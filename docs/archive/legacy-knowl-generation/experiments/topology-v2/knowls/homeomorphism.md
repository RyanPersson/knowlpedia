---
title: "Homeomorphism"
description: "A bijection that is continuous with continuous inverse."
---

A **homeomorphism** is a {{< knowl id="bijective-function" section="shared-foundations" text="bijection" >}} $f:X\to Y$ between {{< knowl id="topological-space" text="topological spaces" >}} such that $f$ is {{< knowl id="continuous-map" text="continuous" >}} and its {{< knowl id="inverse-function" section="shared-foundations" text="inverse function" >}} $f^{-1}:Y\to X$ is also continuous.

If a homeomorphism exists, $X$ and $Y$ are “the same” as topological spaces (they have identical structure of open sets up to relabeling by $f$).

**Examples:**
- The identity map on any space is a homeomorphism.
- The map $f:\mathbb{R}\to\mathbb{R}$ given by $f(x)=2x$ is a homeomorphism (usual topology).
- If $X$ is a set with the discrete topology and $X$ with the indiscrete topology, the identity bijection between them is not a homeomorphism (only one direction is continuous).
