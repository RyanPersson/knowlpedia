---
title: "Continuous image of compact is compact (hence closed and bounded in common targets)"
description: "A continuous image of a compact set is compact, with standard consequences in metric and Euclidean settings"
---

**Corollary (closed/bounded consequences):** Let $f:X\to Y$ be a {{< knowl id="continuous-map" section="topology-core" text="continuous map" >}} and let $K\subseteq X$ be {{< knowl id="compact-set" text="compact" >}}. Then $f(K)$ is compact. If $Y$ is {{< knowl id="hausdorff-space" section="topology-separation" text="Hausdorff" >}}, then $f(K)$ is {{< knowl id="closed-set" section="topology-core" text="closed" >}} in $Y$; if $Y$ is a {{< knowl id="metric-space" section="topology-metric" text="metric space" >}}, then $f(K)$ is {{< knowl id="bounded-set" section="topology-metric" text="bounded" >}}.

This follows from {{< knowl id="continuous-image-of-compact-set-is-compact" text="continuous image of a compact set is compact" >}} together with {{< knowl id="compact-subset-of-hausdorff-is-closed" text="compact implies closed in Hausdorff spaces" >}} and {{< knowl id="compactness-implies-boundedness" text="compactness implies boundedness" >}} in metric spaces.
