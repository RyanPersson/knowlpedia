---
title: "Finite intersection property theorem"
description: "In a compact space, closed sets with the finite intersection property have nonempty intersection"
---

**Finite intersection property theorem:** Let $X$ be a {{< knowl id="compact-set" text="compact" >}} {{< knowl id="topological-space" section="topology-core" text="topological space" >}}. If $\{F_i\}_{i\in I}$ is a family of {{< knowl id="closed-set" section="topology-core" text="closed sets" >}} in $X$ such that every finite intersection $F_{i_1}\cap\cdots\cap F_{i_n}$ is nonempty, then
\[
\bigcap_{i\in I} F_i \neq \varnothing.
\]

This characterization is equivalent to the {{< knowl id="open-cover" text="open-cover formulation" >}} of compactness via complements.
