---
title: "Dense set"
description: "A subset whose closure is the whole space."
---

A **dense set** in a {{< knowl id="topological-space" text="topological space" >}} $X$ is a subset $A\subseteq X$ such that $\overline{A}=X$, where $\overline{A}$ denotes the {{< knowl id="closure" text="closure" >}} of $A$.

Equivalently, $A$ is dense if every nonempty {{< knowl id="open-set" text="open set" >}} $U\subseteq X$ satisfies $U\cap A\neq\varnothing$.

**Examples:**
- $\mathbb{Q}$ is dense in $\mathbb{R}$ with the usual topology.
- $\mathbb{Z}$ is not dense in $\mathbb{R}$.
- In an indiscrete space, every nonempty subset is dense.
