---
title: "Open set"
description: "A set declared open by the topology on a space."
---

An **open set** in a {{{< knowl id="topological-space" text="topological space" >}}} $(X,\tau)$ is a {{{< knowl id="subset" section="shared-foundations" text="subset" >}}} $U\subseteq X$ such that $U\in\tau$.

Open sets are the basic objects used to define notions like {{{< knowl id="neighborhood" text="neighborhood" >}}}, {{{< knowl id="interior" text="interior" >}}}, and {{{< knowl id="continuous-map" text="continuous map" >}}}. In a {{{< knowl id="metric-space" section="topology-metric" text="metric space" >}}}, every {{{< knowl id="open-ball" section="topology-metric" text="open ball" >}}} is open in the {{{< knowl id="metric-induced-topology" section="topology-metric" text="metric-induced topology" >}}}.
