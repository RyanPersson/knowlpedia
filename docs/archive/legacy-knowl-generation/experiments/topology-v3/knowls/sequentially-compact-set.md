---
title: "Sequentially compact set"
description: "A set in which every sequence has a convergent subsequence with limit in the set"
---

A **sequentially compact set** $K$ in a {{{< knowl id="metric-space" section="topology-metric" text="metric space" >}}} $(X,d)$ is a subset such that every sequence in $K$ has a {{{< knowl id="convergent-sequence" section="topology-metric" text="convergent subsequence" >}}} whose {{{< knowl id="limit-of-a-sequence" section="topology-metric" text="limit" >}}} lies in $K$.

Sequential compactness is a “sequence-based” compactness notion; in metric spaces it coincides with {{{< knowl id="compact-set" text="compactness" >}}} (see {{{< knowl id="sequential-compactness-equals-compactness" text="sequential compactness equals compactness" >}}}).
