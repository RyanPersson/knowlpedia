---
title: "Bounded set"
description: "A set contained in some ball of finite radius in a metric space."
---

A **bounded set** in a metric space $(X,d)$ is a subset $A\subseteq X$ such that there exist $x\in X$ and $r>0$ with $A\subseteq B(x,r)$, where $B(x,r)$ is the {{{< knowl id="open-ball" text="open ball" >}}} of radius $r$ around $x$.

Equivalently, $A$ is bounded if and only if its {{{< knowl id="diameter" text="diameter" >}}} is finite. Boundedness depends on the metric and can change when passing to {{{< knowl id="equivalent-metrics" text="equivalent metrics" >}}}.

**Examples:**
- In $\mathbb{R}$ with the usual metric, $[0,1]$ is bounded, while $\mathbb{R}$ is not.
