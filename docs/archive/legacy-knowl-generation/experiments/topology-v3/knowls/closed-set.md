---
title: "Closed set"
description: "A set whose complement is open."
---

A **closed set** in a {{{< knowl id="topological-space" text="topological space" >}}} $(X,\tau)$ is a {{{< knowl id="subset" section="shared-foundations" text="subset" >}}} $F\subseteq X$ such that its {{{< knowl id="complement" section="shared-foundations" text="complement" >}}} $X\setminus F$ is an {{{< knowl id="open-set" text="open set" >}}}.

Closed sets are used to define the {{{< knowl id="closure" text="closure" >}}} of a set as the smallest closed set containing it, and they also provide an equivalent way to check {{{< knowl id="continuous-map" text="continuity" >}}} via preimages.
