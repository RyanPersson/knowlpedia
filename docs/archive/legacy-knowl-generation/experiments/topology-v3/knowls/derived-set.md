---
title: "Derived set"
description: "The set of all limit points of a subset."
---

Let $A\subseteq X$ be a subset of a {{{< knowl id="topological-space" text="topological space" >}}} $X$. The **derived set** of $A$, denoted $A'$, is the set of all {{{< knowl id="limit-point-accumulation-point-cluster-point" text="limit points" >}}} of $A$:
\[
A'=\{x\in X : x \text{ is a limit point of } A\}.
\]

Derived sets are used to describe how $A$ accumulates in $X$, and relate closely to the {{{< knowl id="closure" text="closure" >}}} operation.

**Examples:**
- In $\mathbb{R}$ with the usual topology, if $A=\{1/n : n\in\mathbb{N}\}$ then $A'=\{0\}$.
