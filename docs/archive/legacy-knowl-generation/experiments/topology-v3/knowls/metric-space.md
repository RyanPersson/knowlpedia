---
title: "Metric space"
description: "A set equipped with a metric (distance function)."
---

A **metric space** is an {{{< knowl id="ordered-pair" section="shared-foundations" text="ordered pair" >}}} $(X,d)$ where $X$ is a {{{< knowl id="set" section="shared-foundations" text="set" >}}} and $d$ is a {{{< knowl id="metric" text="metric" >}}} on $X$.

Every metric space carries a natural {{{< knowl id="metric-induced-topology" text="metric-induced topology" >}}}, making it a {{{< knowl id="topological-space" section="topology-core" text="topological space" >}}} whose basic neighborhoods are given by {{{< knowl id="open-ball" text="open balls" >}}}.

**Examples:**
- {{{< knowl id="euclidean-space" section="linear-algebra" text="Euclidean space" >}}} $\mathbb{R}^n$ with the usual Euclidean distance.
- Any set $X$ with the discrete metric (every singleton is open in the induced topology).
