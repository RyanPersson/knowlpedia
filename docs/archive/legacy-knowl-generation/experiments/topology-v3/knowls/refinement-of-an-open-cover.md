---
title: "Refinement of an open cover"
description: "A cover in which every set lies inside some set from a given cover"
---

A **refinement** of an {{{< knowl id="open-cover" text="open cover" >}}} $\mathcal U$ of a set $A\subseteq X$ is an open cover $\mathcal V$ of $A$ such that for every $V\in\mathcal V$ there exists $U\in\mathcal U$ with $V\subseteq U$.

Refinements formalize the idea of “making a cover finer.” They appear naturally in compactness arguments (via {{{< knowl id="finite-subcover-lemma" text="finite subcovers" >}}}) and in metric results such as the {{{< knowl id="lebesgue-number-lemma" text="Lebesgue number lemma" >}}}.
