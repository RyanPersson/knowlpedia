---
title: "Curve"
description: "A continuous map from an interval of real numbers into a topological space."
---

A **curve** in a {{{< knowl id="topological-space" section="topology-core" text="topological space" >}}} $X$ is a {{{< knowl id="continuous-map" section="topology-core" text="continuous map" >}}} $\gamma:I\to X$, where $I\subseteq\mathbb{R}$ is an interval (with the {{{< knowl id="subspace-topology" section="topology-core" text="subspace topology" >}}} from the {{{< knowl id="metric-induced-topology" section="topology-metric" text="usual topology" >}}} on $\mathbb{R}$).

A {{{< knowl id="path" text="path" >}}} is the special case of a curve with domain $[0,1]$, and curves are a basic tool for describing {{{< knowl id="path-connected-set" text="path connectedness" >}}}.
