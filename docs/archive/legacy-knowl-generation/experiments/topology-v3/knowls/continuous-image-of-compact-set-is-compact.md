---
title: "Continuous image of a compact set is compact"
description: "A continuous map sends compact sets to compact sets"
---

**Continuous image of compact is compact:** Let $f\colon X\to Y$ be a {{{< knowl id="continuous-map" section="topology-core" text="continuous map" >}}} between {{{< knowl id="topological-space" section="topology-core" text="topological spaces" >}}}. If $K\subseteq X$ is {{{< knowl id="compact-set" text="compact" >}}}, then the {{{< knowl id="image" section="shared-foundations" text="image" >}}} $f(K)\subseteq Y$ is compact.

This is one of the most frequently used permanence properties of compactness, and it underlies results like {{{< knowl id="continuous-attains-max-min-compact" text="attainment of maxima and minima" >}}}.
