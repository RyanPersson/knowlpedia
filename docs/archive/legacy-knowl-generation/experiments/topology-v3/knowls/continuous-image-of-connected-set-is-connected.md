---
title: "Continuous image of a connected set is connected"
description: "The image of a connected set under a continuous map is connected."
---

**Continuous image of a connected set is connected:** Let $f:X\to Y$ be a {{{< knowl id="continuous-map" section="topology-core" text="continuous map" >}}} between {{{< knowl id="topological-space" section="topology-core" text="topological spaces" >}}}, and let $C\subseteq X$ be {{{< knowl id="connected-set" text="connected" >}}}. Then the set $f(C)$ (the {{{< knowl id="image" section="shared-foundations" text="image" >}}} of $C$ under $f$) is connected in $Y$.

This is a basic permanence property of connectedness and is the key input for the {{{< knowl id="intermediate-value-property-topological" text="topological intermediate value property" >}}} and related interval-image results.
