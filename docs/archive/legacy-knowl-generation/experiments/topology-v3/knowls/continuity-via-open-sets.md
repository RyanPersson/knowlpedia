---
title: "Continuity via open sets"
description: "A map is continuous exactly when preimages of open sets are open."
---

**Continuity via open sets:** Let $X$ and $Y$ be {{{< knowl id="topological-space" text="topological spaces" >}}} and let $f:X\to Y$ be a {{{< knowl id="function" section="shared-foundations" text="function" >}}}. Then $f$ is {{{< knowl id="continuous-map" text="continuous" >}}} if and only if for every {{{< knowl id="open-set" text="open set" >}}} $V\subseteq Y$, the {{{< knowl id="preimage" section="shared-foundations" text="preimage" >}}} $f^{-1}(V)$ is open in $X$.

Equivalently, $f$ is continuous if and only if for every {{{< knowl id="closed-set" text="closed set" >}}} $F\subseteq Y$, the preimage $f^{-1}(F)$ is closed in $X$.
