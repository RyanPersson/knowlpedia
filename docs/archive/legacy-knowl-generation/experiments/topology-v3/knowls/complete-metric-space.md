---
title: "Complete metric space"
description: "A metric space in which every Cauchy sequence converges."
---

A **complete metric space** is a metric space $(X,d)$ in which every {{{< knowl id="cauchy-sequence" text="Cauchy sequence" >}}} converges to a point of $X$ (equivalently, every Cauchy sequence has a {{{< knowl id="limit-of-a-sequence" text="limit" >}}} in $X$).

Completeness is a metric property (not purely topological) and is closely tied to nested closed-set phenomena such as the {{{< knowl id="cantor-intersection-theorem" text="Cantor intersection theorem" >}}}.

**Examples:**
- The real line $\mathbb{R}$ with the usual metric is complete, while $\mathbb{Q}$ with the induced metric is not.
- Any space with the discrete metric is complete, since every Cauchy sequence is eventually constant.
