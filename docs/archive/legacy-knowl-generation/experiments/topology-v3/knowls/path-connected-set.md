---
title: "Path-connected set"
description: "A subset in which any two points can be joined by a continuous path lying entirely in the subset."
---

A **path-connected set** is a subset $A$ of a {{{< knowl id="topological-space" section="topology-core" text="topological space" >}}} $X$ such that for every $x,y\in A$ there exists a {{{< knowl id="path" text="path" >}}} $\gamma:[0,1]\to A$ (continuous for the {{{< knowl id="subspace-topology" section="topology-core" text="subspace topology" >}}} on $A$) with $\gamma(0)=x$ and $\gamma(1)=y$.

Path connectedness is stronger than {{{< knowl id="connected-set" text="connectedness" >}}}: every path-connected set is connected, but not every connected set is path-connected.
