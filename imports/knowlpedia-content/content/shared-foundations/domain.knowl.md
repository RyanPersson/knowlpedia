A **domain** is the set of inputs of a function: if $f:A\to B$ is a [[shared-foundations/function|function]], then its domain is the set $A$.

Changing the domain (for example by taking a [[shared-foundations/restriction-of-a-function|restriction]]) produces a new function, even if the output rule looks the same. The elements of the domain appear as the first components of [[shared-foundations/ordered-pair|ordered pairs]] in the [[shared-foundations/graph-of-function|graph]] of $f$.

**Examples:**
- For $f:\mathbb{R}\to\mathbb{R}$ defined by $f(x)=x^2$, the domain is $\mathbb{R}$ (the [[shared-foundations/real-numbers|real numbers]]).
- For the inclusion map $i:S\to A$ with $S\subseteq A$, the domain is $S$.
