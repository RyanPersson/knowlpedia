A **countable set** is a set $A$ for which there exists an [[shared-foundations/injective-function|injective function]] $f:A\to\mathbb{N}$, where $\mathbb{N}$ is the set of [[shared-foundations/natural-numbers|natural numbers]]. Equivalently, $A$ is countable if it is finite or there exists a [[shared-foundations/bijective-function|bijective function]] between $A$ and $\mathbb{N}$.

Countability is a statement about [[shared-foundations/cardinality|cardinality]]: a set is countable exactly when its size is no larger than that of $\mathbb{N}$. Many constructions in analysis and topology distinguish sharply between countable and uncountable sets.

**Examples:**
- [[shared-foundations/integers|The integers]] $\mathbb{Z}$ form a countable set (they can be arranged in a sequence).
- [[shared-foundations/rational-numbers|The rational numbers]] $\mathbb{Q}$ form a countable set.
