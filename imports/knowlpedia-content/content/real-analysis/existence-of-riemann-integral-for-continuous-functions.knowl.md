**Continuous functions are Riemann integrable**: If $f:[a,b]\to\mathbb{R}$ is [[real-analysis/continuity-on-a-set|continuous]], then $f$ is [[real-analysis/riemann-integrable-function|Riemann integrable]] on $[a,b]$.

This theorem guarantees that the [[real-analysis/riemann-integral|Riemann integral]] covers all standard functions from calculus and is a basic entry point for more refined integrability criteria.
